# PAF-IAST Website Prototype

## Overview
This is a one-page, responsive website prototype designed for the **Pak-Austria Fachhochschule: Institute of Applied Sciences and Technology (PAF-IAST)**. The website serves as an informational hub, detailing the institution's vision, programs, admissions, and contact information.

## Features
- **Responsive Design**: Adapts seamlessly to various screen sizes (desktop, tablet, and mobile).
- **Navigation Menu**: Includes a distinct logo, intuitive navigation links, and a call-to-action "Apply Now" button.
- **Hero Section**: Features an engaging headline and introduction to the institution's goals.
- **About Section**: Highlights the collaborative venture with Austrian and Chinese universities and the goal of skill development.
- **Programs & Innovation**: Showcases key initiatives like the Sino-Pak AI Center, Technology Park, and Industry Internships.
- **Admissions Info**: Details regarding scholarships, exchange programs, and current admission cycles.
- **Contact Form**: Functional front-end contact form with real-time field validation, ready to be connected to a backend or service like Web3Forms.

## File Structure
- `index.html`: The main HTML document containing the structure, content, and layout of the page.
- `style.css`: The CSS stylesheet defining the visual appearance, styling, and responsiveness of the website elements.
- `script.js`: The JavaScript file providing interactivity, smooth scrolling, and form validation logic.

## Setup and Usage
1. Clone or download this repository/folder to your local machine.
2. Open the `index.html` file using any modern web browser (e.g., Chrome, Firefox, Edge, Safari).
3. **Contact Form Setup**: To enable the contact form submission:
   - Go to [Web3Forms](https://web3forms.com/).
   - Generate an Access Key.
   - Open `index.html` in an editor.
   - Replace the `value` attribute `YOUR_ACCESS_KEY_HERE` located in the `<input type="hidden" name="access_key" value="YOUR_ACCESS_KEY_HERE">` field with your actual Web3Forms access key.

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS, variables, flexbox, CSS grid)
- JavaScript (Vanilla JS, DOM manipulation, Form Validation)
- Inter Font (Google Fonts)
